# Serendipity for CodeSandbox.io


# Installation


To change the theme do this:

- Open VS Code
- Press (CMD/CTRL) + Shift + P
- Enter 'Developer: Generate Color Theme From Current Settings'
- Copy the contents
- Paste them in CodeSandbox Preferences


Available themes:

- Morning
- Sunset
- Midnight
